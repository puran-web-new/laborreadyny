# Labor Ready NY Inc Website

This repository contains the static website for Labor Ready NY Inc.

## Deployment

The site is published from the repository root through GitHub Pages.

### Main domain

- Production domain: `https://laborreadyny.xyz`
- Custom domain file: `CNAME`
- Sitemap: `https://laborreadyny.xyz/sitemap.xml`

### IONOS DNS

Point the IONOS apex domain to GitHub Pages with these A records:

- `185.199.108.153`
- `185.199.109.153`
- `185.199.110.153`
- `185.199.111.153`

Optional `www` host:

- `CNAME` `www` → `puran-git-for-web.github.io`
